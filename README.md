# Arduino Night Light - Glow Project (DesInv23)

Interactive night light project using Arduino + p5.js.

## Features
- Photoresistor detects room brightness
- Potentiometer controls orb size and saturation
- LED colors reflect ambient light
- p5.js visualization shows glowing orb and stars

## Hardware
- Arduino Uno
- Photoresistor
- Potentiometer
- Red LED
- Blue LED
- Yellow LED
- Breadboard + resistors

## Software
- Arduino IDE
- p5.js
- Web Serial API

## Folder Structure
- `/arduino` – Arduino code
- `/p5js` – visualization code
